# 微信公众号排版编辑器

转化 Markdown 到给微信特制的 HTML

- 使用在线版： https://lab.lyric.im/wxformat
- 介绍：https://mp.weixin.qq.com/s/pn0LzyfgUj6rGUfVHUksjg

## 优点

- 能较好地排版
- 能解决 `ul` 和 `ol` 元素样式渲染时被微信编辑器重置的问题
- 能把链接自动转换为参考文献索引，并且附在文字末尾
